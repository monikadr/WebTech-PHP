FB search using PHP and FB SDK

Description : 
This is FB Search using FB SDK and PHP to get the json data
This whole program is written in PHP and HTML rendering is done using PHP. 

Download the code

Open "hw6FBSearch_actual_submitted.php" file in local server 

This uses FB authentication so long term access token is need 
if token is expired no worried generate from "https://developers.facebook.com/tools/accesstoken/" and put those tokens to PHP file
